## Apache Jena Framework Architecture
Apache Jena is an open-source and free Java framework designing for building  semantic web and Linked Data application. It provides APIs and tools to handle RDF represntation, query with SPARQL, perform reasoning with OWL/RDFS, and save data in triple store.
